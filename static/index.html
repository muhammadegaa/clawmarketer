Data Cleansing Agent

Scan local files, clean, deduplicate and send reports via Telegram

How it works
1
OpenClaw scans your configured DATA_DIR for CSV/Excel files
2
Each file is cleaned — duplicates removed, nulls fixed, columns normalised
3
Clean files (prefixed clean_) saved back to your machine
4
AI summary + clean files sent to Telegram. Progress shown here.
Telegram triggers
"clean my data" Scans default DATA_DIR
"clean data in ~/reports" Scans custom path
"analyze my files" Same as clean my data
Runs on your machine: Set DATA_DIR in clawmarketer.env to point at your data folder.
