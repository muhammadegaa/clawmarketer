Set up your local agent

    
Run these steps once on your machine. Takes about 5 minutes.


    


      

        
1

        

          
Install OpenClaw

          
OpenClaw runs AI agents locally and connects them to your Telegram bot.

          

            curl -sSL https://openclaw.dev/install.sh | bash
            
          

        

      


      

        
2

        

          
Download your config file

          
Go to Integrations → Download .env (or your profile page) and save it to:

          

            ~/.openclaw/clawmarketer.env
          

        

      


      

        
3

        

          
Install ClawMarketer skills

          
Install the Meta Ads skill (and optionally the Data Cleaner):

          

            openclaw skill install clawmarketer-meta
            
          

        

      


      

        
4

        

          
Start OpenClaw

          
Then send a message to your Telegram bot to trigger your first run:

          
analyze my ads

        

      


    


    

      
Questions? Check the OpenClaw docs.
