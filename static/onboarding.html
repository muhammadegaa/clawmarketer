Setup Guide

Get ClawMarketer running on your machine in under 10 minutes.

Prerequisites

Required Python 3.9+
Required Groq API key (free)
Optional Meta Ads account (for live data)
Optional Firebase project (for auth + history)
1

Clone the repository

Get the code on your machine.

git clone https://github.com/muhammadegaa/clawmarketer.git
cd clawmarketer
2

Install dependencies

Uses Python standard library + a few packages.

pip install -r requirements.txt
3

Configure environment

Copy the example and fill in your keys.

cp .env.example .env

Edit .env with your values:

# Required
GROQ_API_KEY=gsk_...

# For Meta Ads live data
META_APP_ID=...
META_APP_SECRET=...
REDIRECT_URI=http://localhost:8000/auth/callback

# For Firebase auth + history
FIREBASE_API_KEY=...
FIREBASE_AUTH_DOMAIN=...firebaseapp.com
FIREBASE_PROJECT_ID=...
FIREBASE_APP_ID=...
4

Start the server

python3 -m uvicorn server:app --reload --port 8000

Open http://localhost:8000

5

Set up Firebase

Optional

Enables login and saves your report history.

  1. Go to console.firebase.google.com → Create project
  2. Authentication → Get started → Enable Google sign-in
  3. Firestore Database → Create database → Start in test mode
  4. Project Settings → Your apps → Add Web app → copy the config
  5. Paste the values into your .env file
6

Connect Meta Ads

Optional

Required for live campaign data. Skip to use CSV upload or sample data.

  1. Register at developers.facebook.com (use a different email if SMS fails)
  2. Create App → type: Business → add product: Marketing API
  3. Settings → Basic → copy App ID and App Secret
  4. Add http://localhost:8000/auth/callback to Valid OAuth Redirect URIs
  5. Add values to .env → restart server → click "Connect Meta Ads"

Quick test (no Meta account needed)

python3 main.py demo --api-key YOUR_GROQ_KEY

Or just go to the dashboard → Sample Data tab.
