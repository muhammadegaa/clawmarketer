Set up your AI marketing agent

OpenClaw runs on your laptop. You talk to it via Telegram. ClawMarketer shows live results in the dashboard.

1

Install OpenClaw

OpenClaw is an open-source AI agent platform that runs on your machine. Install it with one command:

curl -fsSL https://openclaw.ai/install.sh | bash openclaw onboard

Requires macOS or Linux. The onboard command walks you through initial setup interactively.

Alternatively: npm i -g openclaw if you prefer npm.

2

Create a Telegram bot

Your Telegram bot is how you talk to the agent. It takes 2 minutes to create.

  1. 1. Open Telegram → search @BotFather
  2. 2. Send /newbot → follow prompts → copy the bot token
  3. 3. Find your Telegram user ID (needed for allowlist):
    curl "https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates" Send any message to your bot first, then look for from.id in the response.

Now configure OpenClaw. Edit ~/.openclaw/openclaw.json:

{ channels: { telegram: { enabled: true, botToken: "YOUR_TELEGRAM_BOT_TOKEN", dmPolicy: "allowlist", allowFrom: [YOUR_TELEGRAM_USER_ID] // numeric ID } } }
openclaw restart
3

Add ClawMarketer skills

Skills are directories in ~/.openclaw/skills/. Create two skills — one for Meta Ads, one for data cleansing.

Meta Ads Skill
Data Cleansing Skill
mkdir -p ~/.openclaw/skills/clawmarketer-meta

Create ~/.openclaw/skills/clawmarketer-meta/SKILL.md:

--- name: meta-ads description: Analyze Meta Ads campaigns, generate AI report, send to Telegram user-invocable: true --- ## Meta Ads Analysis Analyze Meta Ads performance via ClawMarketer. ## Configuration Load from ~/clawmarketer.env: - CLAWMARKETER_URL, CLAWMARKETER_USER_ID, META_ACCESS_TOKEN ## Steps 1. Load config: `source ~/clawmarketer.env` 2. Push stage 1 running to $CLAWMARKETER_URL/api/agent/push 3. Call Meta Ads Insights API with META_ACCESS_TOKEN 4. Clean, analyze, generate AI report 5. Push each stage progress to ClawMarketer 6. Send summary + charts to Telegram ## Triggers User says: "analyze my ads", "analyze ads last 30 days", "run ads report"
mkdir -p ~/.openclaw/skills/clawmarketer-data

Create ~/.openclaw/skills/clawmarketer-data/SKILL.md:

--- name: data-cleanse description: Scan local CSV/Excel files, clean, deduplicate, send report user-invocable: true --- ## Data Cleansing Agent Scan and clean local data files via ClawMarketer. ## Configuration Load from ~/clawmarketer.env: - CLAWMARKETER_URL, CLAWMARKETER_USER_ID, DATA_DIR ## Steps 1. Load config: `source ~/clawmarketer.env` 2. Scan DATA_DIR for CSV/Excel files (skip files starting with clean_) 3. For each file: remove duplicates, fix nulls, normalize columns 4. Save clean_ versions in same directory 5. Push progress to ClawMarketer after each file 6. Send AI summary + clean files to Telegram ## Triggers User says: "clean my data", "clean data in ~/reports", "analyze my files"

Place config in home directory:

mv ~/Downloads/clawmarketer.env ~/clawmarketer.env

Clone the skills into OpenClaw workspace:

mkdir -p ~/.openclaw/workspace/skills git clone https://github.com/muhammadegaa/clawmarketer /tmp/clawmarketer cp -r /tmp/clawmarketer/skills/* ~/.openclaw/workspace/skills/

Install Python deps and restart:

cd /tmp/clawmarketer && pip install -r requirements.txt openclaw restart && openclaw skills list

You're set. Try it in Telegram:
