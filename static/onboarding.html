Get started in 3 steps

OpenClaw runs on your machine. ClawMarketer shows you the results. Telegram is how you talk to the agent.

1

Install OpenClaw

OpenClaw is the agent engine that runs on your laptop. It connects to Telegram so you can trigger agents from your phone.

git clone https://github.com/openclaw/openclaw.git cd openclaw npm install

Requires Node.js 18+. Runs locally — agents only work while your laptop is on.

2

Connect your Meta Ads account

Sign in to ClawMarketer and connect Meta Ads. This generates your pre-filled config file.

Sign in first to connect Meta Ads.

Go to dashboard →

You're signed in but Meta Ads is not connected yet.

Connect Meta Ads →
Meta Ads connected

Download your pre-configured config file. It contains your Meta token, Telegram credentials, and ClawMarketer endpoint.

Drop this file in your openclaw/ directory.

Still need to add your Ad Account ID

After downloading, open clawmarketer.env and fill in:

META_AD_ACCOUNT_ID=act_your_account_id

Find it in Meta Ads Manager → the number in the URL after act_

3

Set up Telegram + add ClawMarketer skill

OpenClaw uses a Telegram bot as its interface. Create one, drop in the skill.

Create your Telegram bot

  1. Open Telegram → search @BotFather
  2. Send /newbot → follow prompts
  3. Copy the bot token
  4. Add to clawmarketer.env: TELEGRAM_BOT_TOKEN=your_token

Add ClawMarketer skills to OpenClaw

# In your openclaw/ directory cp path/to/openclaw_agent.py skills/ cp path/to/openclaw_data_agent.py skills/ cp clawmarketer.env .

Add both skills to your OpenClaw config:

{ "skills": [ { "name": "meta_ads", "trigger": ["analyze ads", "run ads report"], "handler": "skills/openclaw_agent.handle" }, { "name": "data_cleanse", "trigger": ["clean my data", "clean data", "analyze my files"], "handler": "skills/openclaw_data_agent.handle" } ] }

Start OpenClaw

npm start

You're done. Try it:

Open Telegram → message your bot:

"analyze ads last 30 days"
"clean my data"

Watch the progress tracker on your ClawMarketer dashboard update in real time.
